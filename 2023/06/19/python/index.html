Python

基础内容

  • 常用的快捷键

    • ctrl + alt +s:打开软件设置
    • ctrl+d：复制当前行代码
    • shift+alt+上\下：将当前的行代码上移或下移
    • ctrl+shift+f10：运行当前代码文件
    • shift+f6:重命名文件
    • ctrl+f：搜索

  • print 语句中加end=’’可以不换行

    • print("hellow",end='')


+ %s 将内容转换为字符串，放入占位位置

+ %d 将内容转换成1整数，放入占位位置

+ %f将内容转换成浮点型，放入占位位置

    • 数字精度控制(和java差不多)

      • 例如%5.2f：表示将宽度控制为5，将小数点精度设置为2
      • %.2f小数点精度设置为2

    • 快速拼接f” {占位内容}”

  • 数据输入

    • input()获取用户输入
    • 获取的数据永远都是字符串类型的

判断语句

  • if （和java略微不同，python里是：结尾例如 if age >18: 满足条件应该做的事）
  • if else（和java相同）
  • if 条件1：elif 条件2： …… else ：

循环语句

  • while

    • while 条件: (其他跟java一样)
    • while循环的循环条件是自定义的，自行控制循环条件
    • for循环是一种“轮询”机制，1是对一批内容进行逐个处理

  • for

    • for 临时变量 in 待处理的数据集：循环满足时执行的代码

    • 函数的传入参数

      • def 函数名(x,y)：
    函数体
    return 返回值
调用函数(a,b)

      • 可以直接用一个变量接受返回值

    • None字面量

      • None是类型‘NoneType’的字面量，用于表示空的、无意义的
      • 不使用return返回none

    • 函数说明文档

      • def add (x,y):
    """

    :param x: 参数x说明
    :param y:参数y的说明
    :return:返回值的说明
    """
    return x+y

add(1,2)

    • 列表下标索引类似java中的数组

      • 从前向后的方向，编号从0开始递增

      • 从后向前的方向，编号从-1开始递减

      • 嵌套列表中的数据例如

        • my_list=[[1,2,3],[2,3,666]]
