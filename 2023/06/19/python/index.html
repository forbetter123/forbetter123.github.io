Python

基础内容

  • 常用的快捷键

    • ctrl + alt +s:打开软件设置
    • ctrl+d：复制当前行代码
    • shift+alt+上\下：将当前的行代码上移或下移
    • ctrl+shift+f10：运行当前代码文件
    • shift+f6:重命名文件
    • ctrl+f：搜索

  • 注释

    • 单行注释

      • 以#开头(一般#号和注释内容一般建议以一个空格隔开)

    • 多行注释

      • 以一对三个双引号引起来来接受说明一段代码的作用和使用方法

  • 数据类型

    • type（）语句可以输出类型信息

  • 数据类型转换

    • str（）转换为字符串
    • int（）转换为整数
    • float（）转换为数字

  • 标识符命名规则

    • 只能出现
      • 英文
      • 中文
      • 数字
      • 下划线
    • 大小写敏感
    • 不可使用关键词

  • 算术运算符

    • /
    • //（取整除）
    • %(取余)
    • **（指数 ）

  • 使用转义字符\接触引号的效用

    • 例如” "111 “输出”111（格式为name=“\“111”）

  • 字符串格式化

    • 字符串拼接%s占位拼接

      • name = "董佬"
message ="卷王 %s" % name
print (message)

    • 也称遍历循环

    • 临时变量，在编程规范上，作用范围，只限定在for内部

      • 实际上可以访问到
      • 但是编程规范上不允许，不建议这么使用

  • range语句

    • range（num）表示获取一个从0开始，到num结束的数字序列不包含num
    • range（num1，num2）从num1到num2不包含num2
      • 例如range（5,10） [5,6,7,8,9]
    • range（num1，num2，step）
      • 例如range（5,10,2） [5,7,9]

  • continue

    • 类似java中的
    • break
      • 类似java中的

补充

  • print 语句中加end=’’可以不换行

    • print("hellow",end='')

  • \t 效果等于 tab键

  • ctrl+/快速注释
